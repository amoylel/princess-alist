协议头部设计¶

0              7             15                           31
+--------------+--------------+----------------------------+
|              |              |                            |
|     type     |     urge     |      sequence number       |
|              |              |                            |
+--------------+--------------+----------------------------+
|                                                          |
|                       data length                        |
|                                                          |
+----------------------------------------------------------+
|                                                          |
|                      data (if any)                       |
|                                                          |
+----------------------------------------------------------+
  1. type：标示数据类型，文字为1，图片为2，GPS为3，确认信息为4。
  2. urge：标示是否为紧急信息（即命令信息），1为真，0为假。
  3. sequence number：序号，用来绑定表单与图片。
  4. data length：数据部分的总长度，不包含头部。

C语言实现参考：

struct gdphdr
{
    unsigned char type,
    unsigned char urge;
    unsigned short seq;
    unsigned int len;
};
